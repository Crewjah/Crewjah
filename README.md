## ğŸ¬ How to Add Your Screenshots

To replace the placeholder screenshots with your actual application screenshots:

1. **Take Screenshots**: 
   - Visit your running application (`npm run dev`)
   - Capture the landing page, dashboard, study timer, etc.
   - Take a screenshot of your GitHub repository stats page

2. **Save with Correct Names**:
   - `landing-hero.png` - Main hero section from landing page
   - `landing-page.png` - Full landing page view
   - `dashboard.png` - User dashboard interface
   - `study-timer.png` - Study timer with custom subjects
   - `github-stats.png` - Your GitHub repository statistics

3. **Add to Public Folder**:
   - Place files in: `client/public/screenshots/`
   - Use exact filenames listed above
   - Recommended size: 800px width for consistency

4. **Commit and Push**:
   ```bash
   git add client/public/screenshots/
   git commit -m "Add application screenshots"
   git push
   ```

The README will automatically display your screenshots!
